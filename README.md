# Markdown Cheat Sheet

---

## I. Basic Syntax

---

### 1. Headings

- # Heading 1 - \# Heading 1
- ## Heading 2 - \## Heading 2
- ### Heading 3 - \### Heading 3
- #### Heading 4 - \#### Heading 4
- ##### Heading 5 - \##### Heading 5
- ###### Heading 6 - \###### Heading 6

---

### 2. Bold

- **Bold text** - \*\*Bold text\*\*

---

### 3. Italic

- _Italicized text_ - \*Italicized text\*

---

### 4. Strikethrough

- ~~The world is flat~~ - \~\~The world is flat\~\~

---

### 4. Blockquote

- > Blockquote - \> Blockquote

---

### 5. Ordered List

1. First item
2. Second item
3. Third item

---

### 6. Unordered List

- First item
- Second item
- Third item

---

### 7. Code

`$ npm install`
